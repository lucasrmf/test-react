## ReactSPA
### Documentation
---
#### Requirements
- Node
- npm

#### Stack
- Node
- Webpack
- ES6 (Babel)
- React
- React Router
- Redux / React Router Redux / Redux Form
- PostCSS + Sass
- Material UI

#### Initialization
```bash
npm i && npm run dev
```

#### Build to production|stage
```bash
ENV=stage/production NODE_ENV=production npm run publish
```
